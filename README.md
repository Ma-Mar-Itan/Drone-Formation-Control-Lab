## рҹ§  Control Strategies

Five fundamentally different answers to: *how do N agents with no global knowledge form a shape?*

### рҹҹЎ Leader-Follower

```
Leader в”Җв”Җв–¶ FollowerвӮҒ в”Җв”Җв–¶ FollowerвӮӮ в”Җв”Җв–¶ ... в”Җв”Җв–¶ FollowerвӮҷ
```

One drone is designated leader. All others maintain rigid relative offsets from it.
Fast to converge, easy to understand.
**Weakness:** Leader dropout collapses the entire formation.

---

### рҹ”ө Consensus-Based

```
xбөў(t+1) = xбөў(t) + Оө В· ОЈвұјвҲҲNбөў [ xвұј(t) вҲ’ xбөў(t) ]
```

Drones exchange state estimates with neighbours and iteratively average toward agreement.
Provably convergent if the communication graph stays connected.
**Strength:** Gracefully degrades when individual drones fail.

---

### рҹҹў Behavior-Based (Reynolds Rules)

```
F_total = wвӮҒВ·F_separation + wвӮӮВ·F_cohesion + wвӮғВ·F_alignment + wвӮ„В·F_goal
```

Inspired by Craig Reynolds' Boids (1987). Four primitive steering behaviours combine into emergent formation. No explicit geometry вҖ” the shape falls out of local rules alone.
**Strength:** Biologically plausible, robust to noise and dropout.

---

### рҹҹЈ Virtual Structure

```
Formation = rigid virtual body  P(t)
Drone i target: pбөў* = P(t) В· offsetбөў
Control: Fбөў = KpВ·(pбөў* вҲ’ pбөў) вҲ’ KdВ·б№—бөў
```

The formation is a single rigid virtual body. Each drone runs a PD controller to track its assigned slot on that body. Smooth, precise, high fidelity.
**Strength:** Best formation accuracy of all five modes.

---

### рҹ”ҙ Potential Field

```
U_attract = ВҪ В· Оҫ В· вҖ–p вҲ’ p_goalвҖ–ВІ
U_repel   = ВҪ В· О· В· (1/ПҒ вҲ’ 1/ПҒвӮҖ)ВІ   for ПҒ < ПҒвӮҖ
F = вҲ’вҲҮU
```

Goal positions pull; drones and obstacles push. Each agent follows the gradient of the combined energy landscape.
**Strength:** Collision avoidance is intrinsic to the control law.
**Weakness:** Local minima can trap drones.

## рҹ”¬ Physics Engine

Each simulation step (`О”t = 50 ms`):

```
1. Force computation     control law + wind + noise
2. Drone repulsion       soft zone, quadratic falloff
3. Obstacle repulsion    wide influence field (4Г— radius + 20 units)
4. Boundary enforcement  soft MAP_EDGE repulsion
5. Acceleration clamp    |F| вүӨ maxAccel
6. Velocity integration  v += F В· О”t
7. Speed clamp           |v| вүӨ maxSpeed
8. Position integration  p += v В· О”t
9. Heading update        Оё = atan2(vy, vx)
10. Collision counting   O(nВІ) pairwise check
```

The main loop uses a **fixed-timestep accumulator** вҖ” physics advances in exact 50 ms chunks regardless of frame rate, preventing time-step instability on slow machines.
